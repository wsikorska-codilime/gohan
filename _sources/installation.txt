==============
Installation
==============

You can download Gohan binary for your platfrom from
github release page.

https://github.com/cloudwan/gohan/releases

Download Binary
----------------

* Download [Gohan Release](https://github.com/cloudwan/ansible-gohan/releases)
* Start server: `./gohan server --config-file etc/gohan.yaml`

Build
----------------

* Install GO >= 1.6
* go get github.com/cloudwan/gohan
